# Assumed Knowledge and tools 

- Overall understanding of Fabric Workspaces
- Fabric Capacity available and running
- Workspace created and placed in this capacity
- Knows how to create an item in a workspace

In the Workspace > New Item > Search Airflow > New Airflow job > Give Name + Create > Open item
